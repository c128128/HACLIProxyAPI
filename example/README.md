## About

This add-on runs [CLIProxyAPI](https://github.com/router-for-me/CLIProxyAPI) inside Home Assistant, allowing you to proxy CLI commands through an API interface.

CLIProxyAPI provides a REST API that allows you to execute command-line operations remotely with authentication and authorization.

## Features

- Easy integration with Home Assistant
- Configurable logging levels
- Secure authentication with OpenAI session keys
- Multi-architecture support
